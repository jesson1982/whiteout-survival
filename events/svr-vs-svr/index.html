Server vs Server (SvS) Strategy

Last Updated: June 2026 | Event Type: Monthly

SvS Overview

SvS is the ultimate PvP event where servers compete for dominance. Coordination is key to victory.

Victory Conditions

  • Control more territories than enemy server
  • Earn higher total points from kills and captures
  • Destroy enemy's central fortress

Coordination Tips

  1. Assign roles: attackers, defenders, scouts
  2. Use voice chat for real-time communication
  3. Focus fire on high-value targets
  4. Rotate troops to maintain pressure
