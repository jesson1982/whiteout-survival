State Expedition Guide

Last Updated: June 2026 | Event Type: Weekly

How It Works

State Expedition is a weekly cooperative event where alliances compete to earn points by completing missions.

Point Farming Strategy

  1. Daily Missions: Complete all 5 daily tasks (500 points)
  2. Alliance Help: Assist members 10 times (200 points)
  3. Resource Gathering: Collect from high-level nodes (300 points)
  4. Enemy Kills: Defeat barbarian camps (400 points)

Reward Priority

  • 1st: Hero Fragments (highest value)
  • 2nd: Speed-ups (time savers)
  • 3rd: Resources (only if needed)
